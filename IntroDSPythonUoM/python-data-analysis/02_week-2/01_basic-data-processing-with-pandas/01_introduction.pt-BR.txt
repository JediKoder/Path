Nesta semana aprofundaremos nossa
investigação sobre como Python pode ser usada para manipular, limpar, e consultar dados
analisando o kit de ferramentas Pandas. O Pandas foi criado por 
Wes McKinney em 2008, e é um projeto de código aberto sob
licença bem abrangente. Como um projeto open source
tem uma comunidade forte, com 100 desenvolvedores de software
todos contribuindo para ajudar a torná-lo um código melhor. Há alguns lugares que você pode visitar
quando quiser buscar ajuda sobre o Pandas. Obviamente, fóruns de discussão do curso
são um grande recurso. Mas não creio que sejam
o melhor recurso para problemas específicos do Pandas. Acho que o melhor recurso é um site
 de perguntas e respostas chamado Stack Overflow. Stack Overflow é usado amplamente
na Comunidade de desenvolvimento de software para postar perguntas sobre programação, 
linguagens de programação e ferramentas de programação. O que é especial sobre o Stack Overflow
 é que ele é fortemente administrado pela comunidade. E a comunidade do Pandas, particularmente,
usa este site como seu principal recurso no auxílio de novos membros. É bem possível, se você postar
uma pergunta no Stack Overflow, e marcá-la como sendo Pandas e relacionada ao Python, que um desenvolvedor Pandas
irá, de fato, respondê-la. Além de servir para postar perguntas,
Stack Overflow é um ótimo lugar para quais situações as pessoas estão se deparando
e como elas podem resolvê-las. Você pode aprender muito passeando
pelas pilhas de perguntas do Stack Overflow. Um segundo recurso que você
pode querer considerar são livros. Em 2012, Wes McKinney escreveu o livro de referência
 definitivo do Pandas chamado "Python for Data Analysis" e publicado
pela editora O'Reilly Media Inc. Como o livro já tem um certo tempo, Wes recentemente anunciou que uma 
nova edição do livro está chegando, e mesmo assim, considero este um bom livro
para se aprender como o Panda funciona. Também gosto do livro, mais breve,
"Learning the Pandas Library" de Matt Harrison. Não é um livro abrangente
sobre análise de dados e estatísticas. Mas se você só quer aprender
o básico de Pandas e quiser fazê-lo rapidamente, 
acho que é um volume bem definido. O campo da ciência de dados
está mudando rapidamente. Há novos kits de ferramentas e métodos
sendo criados todos os dias. Pode ser difícil 
saber tudo disso. Marco Rodriguez e Tim Golden mantém um blog agregador
maravilhoso chamado "Planet Python". Você pode visitá-lo em planetpython.org,
inscreva-se com um leitor de RSS ou, leia os artigos mais recentes
do feed do Twitter @PlanetPython. H´a muitos contribuintes regulares da
ciência de dados Python, e recomendo muito que
siga os feeds RSS. Eis a minha última ficha sobre como
aprofundar seu aprendizado. Kyle Polich conduz um podcast excelente
chamado "Data Skeptic". Bem, não é baseado em Python,
é bem produzido e tem uma maravilhosa mistura de entrevistas
com especialistas na área, assim como aulas educativas curtas. Grande parte do que ele fala é sobre 
 métodos de aprendizagem automática. E se isso é algo que planeja
explorar neste curso de especialização, realmente recomendo
que se inscreva neste podcast. Bem, é isso para a
introdução nesta semana. A seguir, aprofundaremos no Pandas
e falaremos sobre a estrutura de dados "series".
