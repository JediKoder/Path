Bem-vindo a uma introdução à Ciência Dados com Python Este curso é o primeiro curso de cinco em uma grande especialização em Python e Ciência de Dados Cada curso baseia-se progressivamente em seu conhecimento dos cursos anteriores para lhe dar uma visão bem-arredondado do que a Ciência de Dados é, ajudando você a desenvolver habilidades para a prática de Ciência de Dados. A especialização é de um nível médio de dificuldade, e E esperamos que você já tenha estudado o básico de programação e estatística. Nessa especialização estamos focados no ensino de habilidades utilizando a linguagem Python. Existem muitas outras ferramentas que se pode usar em Ciência de Dados, tais como linguagens especializadas de análise estatística como R, ou outras linguagens de propósito geral como Java e C. Nós escolhemos python como base para esta especialização por três razões. Primeiro, é fácil de aprender. Python é atualmente a linguagem escolhida para introdução de estudantes universitários à programação. É usado em oito dos 10 melhores cursos de Ciência da Computação superior dos EUA. Programas em Python tendem a ser menores do que outros que você pode ter visto em outras linguagens, e tem soluções mais naturais para tarefas tipicas que você pode precisar realizar. Se você tem experiência em programação, mas não especificamente em Python, você pode aprender Python muito rapidamente Em segundo lugar, ela é cheia de recursos. Python é uma linguagem de programação muito geral, com um monte de bibliotecas embutidas e se destaca na manipulação de dados, programação de rede e bancos de dados. é madura, e há uma abundância de materiais disponíveis a partir de livros até cursos on-line. Finalmente, Python tem um conjunto significativo de bibliotecas de ciência de dados que se pode usar. A base dessas bibliotecas é chamado de Ecossistema SciPy, e ele ainda tem sua própria série de conferências. Tanto a interface que vamos usar para fazer trabalhos, chamado de Jupiter Notebooks, e as principais bibliotecas para os dois primeiros cursos, Pandas e Matplotlib fazem parte da pilha SciPy. E fornecem uma excelente base para aprendizagem de máquina, mineração de texto e análise de rede. O primeiro curso está dividido em quatro módulos.
